    APP研发平台

    369Cloud基于Hybrid开发模式，使得HTML技术真正具有了作为移动开发平台核心技术的能力，通过采用原生技术进行封装，弥补了HTML 技术的缺陷。Hybrid模式把原生功能封装成组件，以HTML为纽带，通过可视化、可配置化操作，实现从应用创建、自定义认证、组件选择、模板配置、跨 平台编译的一系列开发流程，组合成与原生应用相媲美的移动应用，大大缩短开发周期，节约开发成本，颠覆开发周期冗长的传统模式。

    跨平台移动应用开发引擎，在线生成iOS和Android两个平台的APP。功能模块覆盖了电话、短信、拍照、录音、音视频播放、位置信息、传感器、二维码扫描、第三方支付等应用开发常用的场景。369cloud对核心的API接口进行了封装，方便开发者调用。

    多种类原生组件类别封装，通过JavaScript快速调用组件接口，给予应用快速拼装开发。官方组件拥有高自定义能力，并满足开发者上传自有组件使用。

    跨平台开发环境，混合式开发使用HTML5、CSS3和JavaScript前端技术开发App，轻松完成高体验效果应用的开发、调试、跟踪和模拟。

    集成众多类型App界面模板，帮助开发者快速搭建App界面。并通过多种模板组合完成App整体界面的框架结构。

    APP后端服务

    01

    数据服务

    可视化的云端数据表设计，支持多达10种数据类型，简单、自由地根据业务需求完成数据库配置，轻松实现云与端的数据交互。
    02

    数据统计服务

    实时查看当前应用用户数据，用户行为等等，为业务分析提供数据。用户通过开放的API集成至自己的后台系统。
    03

    消息推送服务

    稳定高效，集成自有推送系统，灵活的推送配置，提供精准用户推送服务。可根据用户的兴趣定期推给用户，帮助开发者高效率地发掘有价值的信息
    04

    云存储服务

    数据对象以 JSON 格式随存随取，数据容量无限扩展；开放的 REST API，支持数据增删改查操作，对图片、音频、视频等二进制文件，提供了弹性存储空间及管理方式。
    05

    业务逻辑服务

    除了现有数据 API 之外，你还可以使用 JavaScript 等多种语言编写私有的业务逻辑，部署到云端，以满足业务场景。

    投资融服务

    投资融服务

    查看更多

    大唐网络投融资服务面向产业互联网和民生互联网，专注于投资对大唐网络资源型移动互联网产业生态建设有利的项目

    重点投资方向：互联网+金融、教育、智能制造、医疗、物流、零售、体育、电商、交通、养老等。 融资服务 　

    1.融资指导：针对企业融资进行整体策划和指导。包括：商业计划书编制、股权结构设计、融资策略制定、融资谈判指导、融资协议风险提示，后续融资路径建议。

    2.创投对接：为创业项目对接369云基金及大唐网络合作创投机构。

    3.资源对接：大唐网络旗下教育、金融、养老等大数据项目资源对接，政府及国企资源对接。 　　

    4.项目路演：为项目安排创业路演，扩大项目影响力，对接创投资本。

    投行业务：

    1.股权融资：大唐网络依靠自有资金和旗下369云基金与数十家签约天使、VC基金合作，对入孵企业进行定向投资。同时，为企业提供商业计划书撰写、项目路演、资本运作规划等咨询服务。

    2.债权融资：与金融机构开展紧密合作，根据初创中小企业的特点，定制债权融资产品，帮助企业解决资金紧张难题，助力企业成长。

    3.资本通道：大唐网络与华融证券等券商战略合作伙伴协力，共同建设创新企业资本通道，形成从新三板到IPO完整的操作体系，助力企业快速进入资本市场，实现企业估值爆发性增长。
